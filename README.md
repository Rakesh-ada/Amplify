🚀 Introducing Amplify - Your Distraction-Free YouTube Experience 🎶
Excited to share my latest project: Amplify, a Chrome Extension built using the YouTube Data APIv3.

🎯 What does Amplify do?
1. Helps you search for any song or video without visiting the YouTube home screen—no more distractions!
2. Add your favorite songs to the extension and access them instantly with just one click.

⚡ Why Amplify?
We’ve all been there—opening YouTube for one song and ending up watching hours of unrelated videos. Amplify keeps you focused and gives you quick access to what matters most.

🛠️ How to Install Amplify in Chrome:
1. Download Amplify from my GitHub repository: https://lnkd.in/eH9pq4sG
.
2. Unzip the folder and locate the "amplify" folder.
3. Open Chrome and go to chrome://extensions/.
4. Enable Developer mode (toggle on the top-right corner).
5. Click on Load unpacked and select the "amplify" folder.
6. Done! The Amplify extension is ready to use. 🎉
